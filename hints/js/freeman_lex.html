Ok, what should I start from?

Actually, you don't need to check, if the first letter is already uppercase. Just take it using indexing and make uppercase with string.toUpperCase() method. 

text[0].toUpperCase()
Clear. What to do next?

The rest of the text remains unchanged, so make a slice from the second letter (remember about 0-based indexing), using string.slice() method. 

text.slice(1)
How to make correct ending of the string?

Let's think. If the string ends with dot already, you should not do anything. If not - add dot. So, how to program such behavior? I offer you to use ternary operator to check if the last character in the string is "." and make respective addition. You also need string.length property. 

text[text.length - 1] == "." ? "" : "."
So, how my final solution should look like?

You just need to wrap your code into function expression, add all strings you've got together and return the final joined string.

Here is the whole solution:

function correctSentence(text: string): string {

    return text[0].toUpperCase() + text.slice(1) + (text[text.length-1]=="."?"":".");
}

Use regular expressions to replace the first character with uppercase version, replace the last dot character with empty string (if there is no dot at the end, it does nothing) and add dot at the end. 

function correctSentence(text: string): string {
    
    return text
        .replace(/^./, x => x.toUpperCase())
        .replace(/\.$/, '')
        + '.';
}

This solution does the same as ours, but using concat() and endsWith() functions. 

function correctSentence(text: string): string {

    return text[0].toUpperCase().concat(text.slice(1), text.endsWith(".") ? "" : ".")
}
