Where should I start solving this mission?

The idea of the mission is that you need to capitalize the first letter of the string. And then check if there is a period at the end of your sentence. If there is, then your string can be returned as the result of the function. If there is no dot - then add it, and return what you got.

What should I do next?

This string method str.upper() returns a copy of the string in which all lowercase letters are converted to uppercase. We only need to convert the first character and to get it we have to use text[0]. 

            text = text[0].upper() + text[1:]

This may be a good start for you.

def correct_sentence(text: str) -> str:

    text = text[0].upper() + text[1:]
    if not text.endswith('.'):
        text += '.'

    return text
