En entrée de votre function est donnée une phrase. Vous devez retourner une copie qui commence toujours par une majuscule et finit par un point.

Faites attention au fait que toutes les corrections n'ont pas nécessairement besoin d'être faites. Si une phrase finit déjà par un point alors en ajouter un autre serait une erreur.

Entrée : Une chaîne de caractères (string).

Sortie : Une chaîne de caractères (string).

Exemples :

{% if interpreter.slug == "js-node" %} 
correctSentence("greetings, friends") == "Greetings, friends."
correctSentence("Greetings, friends") == "Greetings, friends."
correctSentence("Greetings, friends.") == "Greetings, friends."
{% else %} 
correct_sentence("greetings, friends") == "Greetings, friends."
correct_sentence("Greetings, friends") == "Greetings, friends."
correct_sentence("Greetings, friends.") == "Greetings, friends."
{% endif %}

Precondition: Aucun espace en début ou fin de ligne, le texte ne contient que des espaces, des lettres (a-z A-Z), virgules et points.
