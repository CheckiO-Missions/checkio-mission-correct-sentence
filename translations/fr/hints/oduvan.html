Par où commencer pour résoudre cette mission?

L'idée de la mission est que vous devez mettre une majuscule à la première lettre de la chaîne. Ensuite, vous devez vérifier s'il y a un point à la fin de votre phrase. Si c'est le cas, votre chaîne peut être retournée en tant que résultat de la fonction. S'il n'y a pas de point, ajoutez-le et renvoyez ce que vous avez obtenu.

Que dois-je faire ensuite?

Cette méthode string str.upper() renvoie une copie de la chaîne dans laquelle toutes les lettres minuscules sont converties en majuscules. Nous n'avons besoin de convertir que le premier caractère et pour l'obtenir, nous devons utiliser text[0]. 

            text = text[0].upper() + text[1:]

Ceci peut être un bon début pour vous.

Je n'arrive toujours pas à résoudre cette mission... Que dois-je faire maintenant?

La prochaine étape consiste à vérifier si la chaîne de caractères se termine par un point. Vous pouvez utiliser la méthode str.endswith() méthode string: 

If not text.endswith('.'):
    text += '.'

Cette expression vérifie si le texte de la variable text se termine par un point et, si ce n'est pas le cas, ajoute un point à la fin.

def correct_sentence(text: str) -> str:

    text = text[0].upper() + text[1:]
    if not text.endswith('.'):
