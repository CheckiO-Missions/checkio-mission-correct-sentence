Na wejściu Twojej funkcji otrzymasz jedno zdanie. Musisz zwrócić jego poprawną wersję, która rozpoczyna się wielką literą i kończy kropką.

Zwróć uwagę na fakt, że nie wszystkie poprawki są konieczne. Jeśli zdanie kończy się już kropką to dodanie kolejnej będzie traktowane jako błąd.

Wejście: string.

Wyjście: string.

Przykład:

{% if interpreter.slug == "js-node" %} 
correctSentence("greetings, friends") == "Greetings, friends."
correctSentence("Greetings, friends") == "Greetings, friends."
correctSentence("Greetings, friends.") == "Greetings, friends."
{% else %} 
correct_sentence("greetings, friends") == "Greetings, friends."
correct_sentence("Greetings, friends") == "Greetings, friends."
correct_sentence("Greetings, friends.") == "Greetings, friends."
{% endif %}

Warunek wstępny: Brak początkowych i końcowych spacji, tekst zawiera tlko spacje, litery a-z A-Z oraz kropkę.
