Od czego powinienem zacząć rozwiązywanie tej misji?

Misja polega na tym, że należy napisać pierwszą literę ciągu wielką literą. Następnie proszę sprawdzić, czy na końcu zdania znajduje się kropka. Jeśli tak, to ciąg znaków może zostać zwrócony jako wynik funkcji. Jeśli nie ma kropki - proszę ją dodać i zwrócić wynik.

Co powinienem zrobić dalej?

Ta metoda ciągu znaków str.upper() zwraca kopię ciągu znaków, w którym wszystkie małe litery są konwertowane na wielkie. Musimy przekonwertować tylko pierwszy znak i aby go uzyskać, musimy użyć text[0]. 

            text = text[0].upper() + text[1:]

Może to być dla Państwa dobry początek.

Nadal nie mogę rozwiązać tej misji... Co powinienem zrobić dalej?

Następnym krokiem jest sprawdzenie, czy ciąg znaków kończy się kropką. Może Pan użyć metody str.endswith() string: 

If not text.endswith('.'):
    text += '.'

To wyrażenie sprawdza, czy tekst w zmiennej text kończy się kropką, a jeśli nie, dodaje kropkę na końcu.

Proszę mi pomóc połączyć to wszystko w jedno rozwiązanie!

Proszę nie zapominać, że celem CheckiO jest stworzenie funkcji. 

def correct_sentence(text: str) -> str:

    text = text[0].upper() + text[1:]
    if not text.endswith('.'):
        text += '.'

    return text
