Od czego zacząć?
Funkcja str.upper zwraca kopię stringa z wszystkimi znakami zamienionymi na wielkie litery. W tym zadaniu wystarczy przekonwertować tylko pierwszy znak, który można uzyskać za pomocą text[0].
To powinien być dobry punkt startowy.
W jaki sposób mam tego użyć? 
text = text[0].upper() + text[1:]
To powinno zadziałać, spróbuj.
Ale to wciąż nie koniec rozwiązania. Co mam teraz zrobić?
Następnym krokiem jest sprawdzenie czy string kończy się kropką. Możesz użyć funkcji str.endswith
W jaki sposób mam to zrobić? 
if not text.endswith('.'):
    text += '.'
To wyrażenie sprawdza czy tekst kończy się kropką, a jeśli tak nie jest - dodaje kropkę na końcu.
Możesz pomóc mi połączyć to wszystko?

def correct_sentence(text: str) -> str:
    text = text[0].upper() + text[1:]
    if not text.endswith('.'):
        text += '.'
    
    return text
