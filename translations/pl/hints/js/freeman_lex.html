Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

pomysł jest prosty: pierwsza litera wielka, reszta tekstu bez zmian i ewentualnie kropka na końcu.

Ok, od czego powinienem zacząć?

Właściwie nie trzeba sprawdzać, czy pierwsza litera jest już wielka. Proszę po prostu wziąć ją za pomocą indeksowania i zmienić na wielką za pomocą metody string.toUpperCase(). 

text[0].toUpperCase()

Jasne. Co robić dalej?

Pozostała część text pozostaje bez zmian, więc proszę utworzyć wycinek od drugiej litery (proszę pamiętać o indeksowaniu od 0), używając metody string.slice(). 

text.slice(1)

Jak poprawnie zakończyć ciąg znaków?

Proszę się zastanowić. Jeśli ciąg kończy się już kropką, nie należy nic robić. Jeśli nie - proszę dodać kropkę. Jak więc zaprogramować takie zachowanie? Proponuję użyć operatora trójargumentowego, aby sprawdzić, czy ostatnim znakiem w ciągu jest "." i wykonać odpowiednie dodanie. Potrzebują Państwo również właściwości string.length. 

text[text.length - 1] == "."? "": "."

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy zawinąć kod w wyrażenie funkcyjne, dodać wszystkie ciągi znaków i zwrócić końcowy połączony ciąg.

Oto całe rozwiązanie:

function correctSentence(text: string): string {

    return text[0].toUpperCase() + text.slice(1) + (text[text.length-1]=="."?"":".");
}

Jasne, proszę pobrać!

Proszę użyć wyrażeń regularnych, aby zastąpić pierwszy znak wersalikami, zastąpić ostatni znak kropki pustym ciągiem (jeśli nie ma kropki na końcu, to nic nie robi) i dodać kropkę na końcu. 

function correctSentence(text: string): string {
    
    return text
        .replace(/^./, x => x.toUpperCase())
        .replace(/\.$/, '')
        + '.';
}

To rozwiązanie robi to samo co nasze, ale używa funkcji concat() i endsWith(). 

function correctSentence(text: string): string {

    return text[0].toUpperCase().concat(text.slice(1), text.endsWith(".")? "": ".")
}
