💡想法很简单：把第一个字母变成大写，其他文字不变，必要时在末尾加点。

好吧，我应该从什么开始？

实际上，你不需要检查第一个字母是否已经是大写字母。只需使用索引，并使用string.toUpperCase() 方法将其变为大写。 

text[0].toUpperCase()

清楚了。接下来该怎么做？

text 的其余部分保持不变，所以从第二个字母开始做一个切片（记住关于基于0的索引），使用string.slice() 方法。 

text.slice(1)

如何使字符串的结尾正确？

让我们想想。如果字符串已经以点结束，你不应该做任何事情。如果没有--添加点。那么，如何对这种行为进行编程呢？我建议你使用三元运算符来检查字符串中的最后一个字符是否是"."，并进行相应的添加。你还需要string.length 属性。 

text[text.length - 1] == "."? "": "."

那么，我的最终解决方案应该是什么样子的呢？

你只需要把你的代码包装成函数表达式，把你得到的所有字符串加在一起，然后返回最后的连接字符串。

下面是整个解决方案：

function correctSentence(text: string): string {

    return text[0].toUpperCase() + text.slice(1) + (text[text.length-1]=="."?"":".");
}

使用正则表达式将第一个字符替换为大写版本，将最后一个点字符替换为空字符串（如果末尾没有点，则不做任何处理），并在末尾添加点。 

function correctSentence(text: string): string {
    
    return text
        .replace(/^./, x => x.toUpperCase())
        .replace(/\.$/, '')
        + '.';
}

这个解决方案与我们的相同，但使用concat() 和endsWith() 函数。 

function correctSentence(text: string): string {

    return text[0].toUpperCase().concat(text.slice(1), text.endsWith(".")? "": ".")
}
