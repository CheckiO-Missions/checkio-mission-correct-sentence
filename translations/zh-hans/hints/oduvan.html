我应该从哪里开始解决这个任务？

这个任务的思路是，你需要将字符串的第一个字母大写。然后检查你的句子末尾是否有句号。如果有，那么你的字符串就可以作为函数的结果返回。如果没有句号--那么就加上它，然后返回你得到的结果。

接下来我应该做什么？

这个字符串方法 str.upper()返回一个字符串的副本，其中所有小写字母都被转换为大写。我们只需要转换第一个字符，为了得到它，我们必须使用text[0] 。 

            text = text[0].upper() + text[1:]

这对你来说可能是一个好的开始。

def correct_sentence(text: str) -> str:

    text = text[0].upper() + text[1:]
    if not text.endswith('.'):
        text += '.'

    return text
