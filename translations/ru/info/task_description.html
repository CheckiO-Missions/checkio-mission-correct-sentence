На вход Вашей функции будет передано одно предложение. Необходимо вернуть его исправленную копию так, чтобы оно всегда начиналось с большой буквой и заканчивалось точкой.

Обратите внимание на то, что не все исправления необходимы. Если предлоедние уже начинается с точки, то добавлять еще одну не нужно, это будет ошибкой

Входные аргументы: String.

Выходные аргументы: String.

Примеры:

{% if interpreter.slug == "js-node" %} 
correctSentence("greetings, friends") == "Greetings, friends."
correctSentence("Greetings, friends") == "Greetings, friends."
correctSentence("Greetings, friends.") == "Greetings, friends."
{% else %} 
correct_sentence("greetings, friends") == "Greetings, friends."
correct_sentence("Greetings, friends") == "Greetings, friends."
correct_sentence("Greetings, friends.") == "Greetings, friends."
{% endif %}

Предусловия: в начале и конце нет пробелов, текст состоит только из пробелов, a-z A-Z , и .
