З чого мені розпочати вирішувати цю місію?
Ідея місії в тому, що тобі потрібно зробити першу літеру рядку великою. А далі перевірити чи в кінці твого речення є крапка. Якщо є, то твій рядок можна повертати як результат функції. Якщо крапки немає - тоді додай її, та поверни те що в тебе вийшло.
Що мені робити далі?
цей метод str.upper() повертає копію рядка в якому всі літери нижнього регістру переведенні у верхній регістр. Нам потрібно перетворити лише перший символ, і щоб отримати його, ми повинні використовувати text[0]. 
text = text[0].upper() + text[1:]

Це може бути хорошим початком для тебе.
Я все ще не можу вирішити цю місію... Що мені протрібно зробити далі?
Наступним кроком для тебе буде перевірка на те, чи закінчується рядок крапкою. Ти можеш використати рядковий метод str.endswith() 
If not text.endswith('.'):
    text += '.'
Цей вираз перевіряє, чи закінчується текст у змінній "text" крапкою, і якщо ні, додає крапку в кінці.
Допоможи мені об'єднати це все в одне рішення!
Не забувай, що твоя мета на CheckiO — створити функцію. 
def correct_sentence(text: str) -> str:
    text = text[0].upper() + text[1:]
    if not text.endswith('.'):
        text += '.'

    return text
