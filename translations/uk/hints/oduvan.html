З чого мені розпочати вирішувати цю місію?
Ця функція str.upper повертає копію рядка в якому всі літери нижнього регістру переведні у верхній регістр. Нам потрібно перетворити лише перший символ, і щоб отримати його, ми повинні використовувати text[0]
Це може бути хорошим початком для тебе.
Як мені цей код використати? 
text = text[0].upper() + text[1:]
Це має працювати як треба, спробуй.
Я все ще не можу вирішити цю місію... Що мені протрібно зробити далі?
Наступним кроком для тебе буде перевірка на те, чи закінчується рядок крапкою. Ти можеш використати функцію str.endswith
Як мені це зробити? 
If not text.endswith('.'):
    text += '.'
Цей вираз перевіряє, чи закінчується текст у змінній "text" крапкою, і якщо ні, додає крапку в кінці.
Допоможи мені об'єднати це все в одне рішення!
Не забувай, що твоя мета на CheckiO — створити функцію. 
def correct_sentence(text: str) -> str:
    text = text[0].upper() + text[1:]
    if not text.endswith('.'):
        text += '.'

    return text
