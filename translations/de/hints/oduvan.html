Wo soll ich mit der Lösung dieser Mission beginnen?

Der Sinn der Aufgabe besteht darin, dass Sie den ersten Buchstaben der Zeichenkette groß schreiben müssen. Und dann prüfen Sie, ob am Ende des Satzes ein Punkt steht. Wenn ja, dann kann Ihre Zeichenkette als Ergebnis der Funktion zurückgegeben werden. Wenn kein Punkt vorhanden ist, fügen Sie ihn hinzu und geben Sie zurück, was Sie erhalten haben.

Was soll ich als nächstes tun?

Diese String-Methode str.upper() gibt eine Kopie der Zeichenkette zurück, in der alle Kleinbuchstaben in Großbuchstaben umgewandelt sind. Wir müssen nur das erste Zeichen umwandeln und dazu müssen wir text[0] verwenden. 

            text = text[0].upper() + text[1:]

Dies könnte ein guter Anfang für Sie sein.

Ich kann diese Mission immer noch nicht lösen... Was soll ich als nächstes tun?

Als nächstes müssen Sie prüfen, ob die Zeichenfolge mit einem Punkt endet. Sie können die str.endswith() string-Methode verwenden: 

If not text.endswith('.'):
    text += '.'

Dieser Ausdruck prüft, ob der Text in der Variablen text mit einem Punkt endet, und fügt andernfalls einen Punkt an das Ende an.

Helfen Sie mir, das alles in einer Lösung zusammenzufassen!

Vergessen Sie nicht, dass Ihr Ziel bei CheckiO darin besteht, eine Funktion zu erstellen. 

def correct_sentence(text: str) -> str:

    text = text[0].upper() + text[1:]
    if not text.endswith('.'):
        text += '.'

    return text
